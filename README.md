# 夏令营日程助手

面向保研夏令营、预推免申请的 Windows 桌面日程助手。它把分散在学校官网、公众号、报名系统里的报名时间、公布时间、开营时间和链接整理到一个直观日历里，适合同时关注多所学校的同学使用。

## 主要功能

- 记录学校、学院、报名号、通知链接、报名时间、公布时间、开营时间、活动形式、地址和备注。
- 日历用不同颜色区分待确认、报名、报名截止、公布、开营等节点。
- 项目列表按接下来真正需要关注的时间排序，过期和放弃/落选项目自动靠后。
- 支持关注学校标记，重点项目带星标，扫一眼就能看出来。
- 支持 AI 一键读取通知链接或通知正文，自动提取学校、学院、报名时间、报名网址、公布时间、开营时间和备注。
- 支持 Excel 备份导入/导出，换电脑或重装软件前可以保存数据。

## AI 解析

软件支持兼容 OpenAI Chat Completions 格式的接口。你可以填写自己的 API 地址、模型名和 API Key。配置完成后，粘贴学校通知链接，软件会抓取网页正文并交给 AI 整理成表单字段。

## 数据说明

你的日程数据保存在本机。AI 解析时，通知正文会发送给你配置的 AI 服务商；不想发送的内容可以手动录入。

## 使用建议

建议每隔几天使用“导出备份”保存一份 Excel 文件。尤其是在大量修改学校信息、换电脑或重装系统前，先导出备份会更稳。

## 适用场景

适合正在准备保研夏令营、预推免、导师联系和多校报名管理的本科生使用。
